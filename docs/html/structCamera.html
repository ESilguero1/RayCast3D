Detailed Description

Camera state structure (all values in Q16.16 fixed-point).

Contains the player's position in the tile grid, a unit-length direction vector indicating where the camera faces, and a camera plane vector perpendicular to the direction that defines the field of view. The plane's magnitude controls the FOV width (default 0.66, giving roughly a 66-degree horizontal FOV).

Access via Camera_Get(). Do not modify directly — use the Camera_Set / Camera_Move / Camera_Rotate functions instead.
