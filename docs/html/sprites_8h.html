Detailed Description

RayCast3D Sprite Rendering Module - Billboard sprites with depth sorting.

Author
Elijah Silguero (with contributions from Surya Balaji)
Date
December 2025

Provides billboarded sprite rendering integrated with the raycasting Z-buffer for proper depth occlusion.
