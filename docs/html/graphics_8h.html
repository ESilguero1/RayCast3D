RayCast3D Graphics Library - Core raycasting and rendering.

Date

Provides the main rendering interface including raycasting, sprite rendering, FPS display, and text overlay functions.

Queue a foreground sprite for rendering on the current frame.

Call before RayCast3D_Render() each frame you want sprite displayed. Queue is automatically cleared after RayCast3D_Render() completes. Max 8 sprites.

Parameters
imagePointer to 16-bit color image data
xScreen X position
yScreen Y position (bottom of sprite)
widthSource image width in pixels
heightSource image height in pixels
scaleScale factor (scale/8 * SCREEN_HEIGHT)
transparentColor key that will not be drawn
