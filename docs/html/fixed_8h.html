Detailed Description

RayCast3D Fixed-Point Math Library (Q16.16)

Author
Elijah Silguero
Date
December 2025 @hardware MSPM0G3507

Range: -32768.0 to +32767.99998 with precision ~0.00002 Includes lookup tables for fast sin/cos and reciprocal.
