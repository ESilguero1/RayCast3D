Detailed Description

Top-level engine initialization and per-frame rendering.

The Engine Core provides the two entry points for the RayCast3D library. Call RayCast3D_Init() once at startup to configure the display and DMA hardware, then call RayCast3D_Render() once per iteration of your main loop to cast rays, draw sprites and overlays, and transfer the frame to the ST7735 display via DMA.

Initialize the raycasting engine.

Configures the ST7735 display, SPI bus, and DMA channel 0 for asynchronous frame transfers. Must be called once at startup before any other RayCast3D function.

void RayCast3D_Render ( void  )

#include <raycast3d.h>

Render one complete frame to the display.

Performs the full rendering pipeline for each of the four screen quarters: clears the depth buffer, casts rays to draw textured walls, renders world sprites with depth sorting, draws queued UI overlays (text, foreground sprites, FPS counter), and transfers each quarter to the ST7735 via DMA. Call once per iteration of your main loop.
