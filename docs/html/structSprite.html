Detailed Description

World sprite descriptor.

Each sprite occupies a slot in the global Sprites_Array. It holds a world-space position, a pointer to its BGR565 image data, and rendering parameters (dimensions, scale, transparency key). The raycaster projects the sprite into screen space as a billboard and depth-tests it against the Z-buffer each frame.

The type field is not used by the engine — it is available for your game logic (e.g., to distinguish enemies from items).
